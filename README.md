# TrashSystem

Making a trash system for my server. Incrementally.

Right now, you can craft a trash can. That's it, though. Open an issue if you think the recipe is ugly!

## Commands

| Command                              | Purpose                   |
|--------------------------------------|---------------------------|
| /trashcan                            | Base command              |        
| /trashcan reload                     | Reload config.yml         |        
| /trashcan set <small\|medium\|large> | Make any item a trash can |
